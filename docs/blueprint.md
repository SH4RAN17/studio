# **App Name**: Sharan Belel

## Core Features:

- Portfolio Landing Page: Displays a hero section with a background image, navigation, and a welcome message.
- About Me Section: Presents a brief introduction with a two-column layout containing a title and a description of the artist.
- Selected Projects Grid: Shows a grid of selected projects with thumbnails, titles, and category labels.
- Project Pages: Dedicated pages for each project, featuring a full-screen render, project details, image gallery, and descriptions.
- Resume Page: Displays a detailed resume with sections for education, skills, services, and contact information.
- Firebase Hosting: Configures Firebase hosting for website deployment, including setting up the firebase.json configuration and organizing the public folder structure.

## Style Guidelines:

- Background color: Near black (#0A0A0A) for a premium dark minimal style.
- Section background: Dark gray (#111111) to provide contrast.
- Primary text color: White (#FFFFFF) for clear readability.
- Secondary text color: Medium gray (#A0A0A0) for less critical information.
- Accent color: Soft warm beige (#E6C8A0) for highlights and key elements.
- Headings font: 'Playfair', a serif font, with sizes between 110-140px for H1 and 60-70px for H2.
- Body font: 'Inter', a sans-serif font, with a size of 18-20px.
- Large cinematic hero sections with full-screen background images and dark gradient overlays.
- Left-aligned headings and right-aligned paragraph blocks for a clean, structured look.
- Grid-based project layout with a 3-column grid for displaying selected projects.
- Smooth scroll animations for a seamless browsing experience.
- Smooth hover effects on project cards and elegant fade transitions for a visually appealing user experience.
